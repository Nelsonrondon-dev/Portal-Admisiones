# Portal Admisiones

Web portal made to measure made in Laravel 7. For the company EADIC, to support its admission system.

## APP

[Portal-Admisiones](https://eadic.org/portal-admisiones/public/login)


## Tech Stack

**Login Access:** Google auth, Facebook auth, Linkedin auth

**Client:** Bootstrap, Js, Jquery, Moment.js, Calendar.js, Dropzone.js

**API** Vtiger CRM

**Server:** PHP, Laravel 7, MySQL.
